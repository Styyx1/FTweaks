# DLLTweaks

* [Nexus Page](https://www.nexusmods.com/skyrimspecialedition/mods/)

### Requirements

* [XMake](https://xmake.io/) [2.7.8+]
* C++23 Compiler (MSVC)

## Build
```
xmake build
```
